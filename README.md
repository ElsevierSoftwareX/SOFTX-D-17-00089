Welcome to the QALMA github repository. QALMA stands for Quality Assurance for Linac with Matlab. It is a Matlab based toolkit with GUI for quantitative analysis of Quality Assurance tests for Medical Linear Accelerators in Radiation Therapy. It is developed at Florida Atlantic University by M. Mushfiqur Rahman under supervision of Dr. Georgios Kalantzis. Currently, versoion 1.0 is available for download. Version 1.0 performs QA tests: 

* Star Shot
* Picket Fence
* Winston-Lutz 
* Log File Analysis 
* Light-radiation field coincidence. 

Sample images and files are available in the ‘Sample’ folder. For detail documantation Please see the manual ('qalma_manual.pdf') or https://github.com/mrmushfiq/qalma/wiki
